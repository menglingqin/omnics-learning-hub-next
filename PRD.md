# OmniCS Learning Hub 产品需求文档 (PRD)

## 1. 产品概述与核心定位

### 1.1 产品愿景
**OmniCS Learning Hub (CS.Academy)** 是一款专为下一代计算机科学家和高级软件工程师打造的沉浸式在线学习平台。平台平衡了学术严谨性与现代工程实践，通过提供深度的计算机科学理论模块和完全交互式的在线实验室环境，帮助用户真正理解底层的“机器本质”，而非仅仅掌握应用编写。

### 1.2 核心目标用户
* **进阶软件工程师**：希望向底层架构、分布式系统、高性能计算等方向转型的开发者。
* **计算机专业学生/求职者**：为顶级科技公司技术面试做准备，追求算法深度与系统设计思维的学习者。

### 1.3 产品定位与风格
* **极致的极客美学 (Technological Minimalism)**：采用深色背景降低长时间编码的眼部疲劳，辅以高饱和度的电极蓝 (Electric Cyan) 和蓝紫色作为指示性交互高亮。使用 **Tonal Layering** 和 **Glassmorphism**（玻璃拟物化）增强界面深度，配合流畅的微交互，传递高端、专业、深邃的极客质感。

## 2. 视觉设计与设计系统 (Design System)

### 2.1 配色方案 (Color Scheme)
* **基础背景色 (Primary Background)**: `#0A192F` 极客深空蓝，用于底盘。
* **高亮激活色 (Secondary/Active)**: `#41E4C0` (或 `#64FFDA`) 电极青色，用于高亮状态、主操作按钮及进度条。
* **卡片及容器色 (Tertiary Surface)**: `#112240` 用于卡片背底、侧边栏及辅助区域。
* **辅助暗底色 (Recessed Code Background)**: `#020C1B` 用于代码编辑区及控制台输出底板。
* **主文本色 (High Emphasis)**: `#F8FAFC` 近纯白。
* **次级文本色 (Secondary Text)**: `#8892B0` / `#C5C6CD` 灰蓝，用于描述和辅助标签。

### 2.2 字体体系 (Typography)
* **主工作字体 (UI Text)**: `Inter` 用于全局UI描述及导航，提供高清晰度与现代感。
* **技术/代码字体 (Technical Text)**: `JetBrains Mono` 用于代码编辑区、控制台、数据统计指标及辅助元数据（如 `n = 45` 或标签芯片），明确区分理论阅读与技术实践。

### 2.3 形状与修饰 (Shapes & Decoration)
* **控制控件/圆角**: 
  * 标准按钮/输入框: `4px` 圆角 (Precise & Architect)
  * 卡片/代码编辑器: `8px` 圆角
  * 进度条: `9999px` (Pill 药丸圆角)
* **发光动效**: 对 Hover 态卡片和激活态按钮使用 `box-shadow: 0 0 15px rgba(65, 228, 192, 0.25)` 进行外发光模拟。

## 3. 功能模块需求 (Functional Requirements)

### 3.1 导航模块 (Navigation & Global Layout)
* **双语切换 (i18n)**: 默认展示中文。右上角配备 `EN / 中文` 切换控件，支持全站翻译的即时切换。
* **自适应活动指示**: 导航栏应支持检测当前活动路由（如 `/courses`, `/lab`, `/careers`），并对活动路由添加电极蓝底边框及高亮效果。
* **实时全局搜索框**: 支持输入关键字实时高亮/检索内容。

### 3.2 首页 (Home Page)
* **Hero 介绍区**: 展示核心 Slogan（“掌握未来的架构”）。右侧放置流线型的动态代码环 (Conceptual Abstract Visualization)，模拟 IDE 代码初始化渲染动效。
* **平台核心指标**: 动态数据卡片展示，涵盖 142K+ 活跃学员、8.5M 累计代码行数、450+ 互动实验及 94% 就业率。
* **精选学习路径 (Curated Paths)**:
  * 核心系统设计路径：包含 12 模块、40 小时内容，展示“微服务与分布式共识”。
  * 机器学习路径：基础数学到 LLM 微调。
  * 现代 Web 工程路径：高级边缘计算与前端架构。
* **热门课程推荐**: 精选 4 门热门课程卡片，支持进度条显示和一键“继续实验”。

### 3.3 课程目录 (Course Library)
* **多维度过滤器侧边栏 (Filter Sidebar)**:
  * **难度等级**: 初级 (Beginner)、中级 (Intermediate)、高级 (Advanced)，支持多选。
  * **学习主题**: 算法 (Algorithms)、数据结构 (Data Structures)、机器学习 (Machine Learning)、基础理论 (Fundamentals)，支持多选。
  * **编程语言**: 支持单选/多选 Python、C++、JS、Go 过滤。
  * **清除过滤器**: 一键将所有选项重置，并展示完整列表。
* **智能检索逻辑**: 课程目录列表卡片需根据侧边栏的多重选定条件，在前端自动执行组合过滤，并支持无匹配结果时的缺省状态提示。
* **多样化课程卡片渲染**:
  * “进行中 (In Progress)”卡片：高亮边框，展示学习百分比进度条（如 68%），包含“继续实验”行动点。
  * “已完成 (Completed)”卡片：模糊遮罩，显示已通关勾选图标，包含“重新复习”行动点。

### 3.4 职业生涯中心 (Career Center)
* **求职职位公告板 (Job Board)**:
  * 包含招聘岗位卡片，展示岗位名、公司、工作地点（Remote/On-site）、技术栈需求。
  * 支持 “REMOTE” 与 “ON-SITE” 按钮的类型过滤切换。
* **简历构建器 (Resume Builder)**:
  * 允许用户点击“编辑简历”按钮拉起全屏或模态弹窗构建器。
  * 提供表单项：姓名、邮箱、GitHub链接、求职岗位、专业技能、项目经历。
  * **实时编译与预览**: 随着输入更新，右侧以 ATS 友好格式实时排版预览简历，并提供打印/导出服务。
* **模拟面试系统 (Mock Interviews)**:
  * 提供预设高频挑战项目，如 “动态规划 (HARD)” 和 “图遍历 (MEDIUM)”。
  * 点击“开始会话”按钮可深度连接至实验平台，自动载入所选择的匹配编程题目及初始代码模板。
* **企业洞察板块 (Company Insights)**:
  * 提供顶级科技公司的面试指南、薪资股权等基础知识板块。

### 3.5 交互式代码实验平台 (Active Lab Workspace)
* **任务指导面板 (Instruction Panel)**:
  * 提供“任务说明 (Instructions)”与“视频课程 (Lesson)”选项卡切换。
  * 包含题目说明、数学公式表示、示例及约束边界条件。
  * 提供“查看提示”功能，弹出相关算法思路的辅助警告提示。
* **代码编辑器面板 (Code Editor Panel)**:
  * **可交互编辑**: 替换原有的静态展示。编辑器为可编写输入区，行号根据换行符 `\n` 自动生成。
  * **自适应代码模版**: 根据从求职页面或课程页面点进的题目，初始化为对应的 Python 算法模版（默认载入“爬楼梯”模版）。
* **控制台测试面板 (Console Panel)**:
  * **多测试用例切换**: 提供 Case 1 / Case 2 / Case 3 等快捷按钮，点击后自动更新 Input（输入参数）和 Expected Output（预期输出）字段。
  * **代码执行功能 (Run Code)**: 模拟/实际检测代码。点击“运行代码”后触发 Load 状态，控制台输出“运行测试中...”。延迟结束后根据用户当前填写的 input 计算出输出，并在控制台高亮输出“✔ 测试用例 X 通过！”或错误信息。
  * **提交验证功能 (Submit)**: 点击“提交”，显示“提交中...”，成功后弹出 Alert 框告知用户通过全部测试，并将该题状态同步为“已完成”。

## 4. 非功能性需求

* **极致的用户体验与流畅动画**: 包含过渡延迟以模拟编译过程，使平台更显“沉浸式”与“专业”。
* **高质量响应式适配**: 移动端折叠导航栏，侧边栏能够自动隐藏/展开。
* **代码健壮性与完整性**: 不使用 Placeholders。测试用例需要全面覆盖新增的交互逻辑。
